# hw-flash / atlantis-deepsea

A collection of hardware‑flashing utilities and device research for LilyGo / ESP‑based boards (e.g., for flashing modems and experimenting with serial / UART / flash tools).

This project is **open source** and released under the [MIT License](LICENSE).

## What this is

- Scripts and tools for flashing ESP32‑based LilyGo devices.
- Example or research code for device‑specific configurations (e.g., modem, GPS, LTE modules).
- Intended for learning, debugging, and low‑level experimentation.
